# Kitchen Background Images

This directory needs two background images for the Cooking Support System:

## Required Images:

### 1. kitchen-light.png
**Description:** Modern 3D kitchen scene with daylight
- **Lighting:** Natural daylight coming through windows
- **Style:** Clean, modern kitchen design
- **Colors:** Bright, white/cream tones, natural wood
- **Mood:** Fresh, inviting, daytime atmosphere
- **Recommended size:** 1920x1080 or higher
- **Format:** PNG for transparency support

**Suggested elements:**
- Modern kitchen cabinets (white or light wood)
- Marble or granite countertops
- Stainless steel appliances
- Large windows with natural light
- Fresh herbs or plants
- Clean, organized workspace

### 2. kitchen-dark.png
**Description:** Modern 3D kitchen scene with artificial lighting
- **Lighting:** Warm artificial lighting (pendant lights, under-cabinet LEDs)
- **Style:** Same modern kitchen as light mode, but evening atmosphere
- **Colors:** Warm amber/orange tones from lighting, darker shadows
- **Mood:** Cozy, intimate, evening atmosphere
- **Recommended size:** 1920x1080 or higher
- **Format:** PNG for transparency support

**Suggested elements:**
- Same kitchen layout as light mode
- Warm pendant lights over island/counter
- Under-cabinet LED strips
- Soft ambient lighting
- Evening/night atmosphere
- Possibly some cooking in progress

## How to obtain these images:

### Option 1: AI Image Generation
Use AI tools like:
- Midjourney: `/imagine modern 3D kitchen with natural daylight, photorealistic, clean design`
- DALL-E 3: "Photorealistic 3D render of a modern kitchen with natural daylight"
- Stable Diffusion: Similar prompts

### Option 2: Stock Photos
Search on:
- Unsplash.com
- Pexels.com
- Pixabay.com
Keywords: "modern kitchen interior", "3D kitchen render", "luxury kitchen"

### Option 3: 3D Rendering
Use tools like:
- Blender (free)
- SketchUp
- 3ds Max

## Installation:
1. Place `kitchen-light.png` in this directory
2. Place `kitchen-dark.png` in this directory
3. The CSS is already configured to use these images
4. Refresh the browser to see the changes
